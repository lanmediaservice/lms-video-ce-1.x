Установка:
1. Скопировать каталог announce в каталог скрипта

2. Установить права на запись для слудующих папок и файлов: 
trailers/
images/
data.inc.php

3. Переименовать config.dist.php в config.php и отредактировать

4. Переименовать шаблон и добавить в самый конец файла header.php:
<div id='announce_box' style='width:100%;border:1px solid silver;margin-top:3px;'>
<a href='javascript:Hide("announce_box")'><img src='images/delete_16.gif' border=0 style='float:right;margin-right:3px;margin-top:3px;background:#F5F5F5;'></a>
Ожидаемые фильмы:
<iframe style='margin-top:3px;' scrolling=no frameborder=0 src="announce/announce.php" width="100%" height="175px"></iframe>
</div>


Добавление трейлеров
1. Добавить новый(е) трейлер(ы) можно двумя способами:
    а) скопировать в каталог trailers/ и в Панеле управления
       в разделе "Неучтенные трейлеры" добавить кнопкой "[^]"  
    б) Нажать кнопку "Добавить фильм", затем указать все поля


Замечания по редактированию

    При вводе в поле "Картинка" не локального пути, она будет автоматически
закачана, скопирована в папку определенную настройками и уменьшена до высоты в 105 пикселей

    Картинку можно загрузить с локального компьютера. Для этого предусмотрено
поле "Новая картинка"   

    Трейлер также как и картинку можно загрузить с локального компьютера. Для этого предусмотрено
поле "Новый трейлер"
